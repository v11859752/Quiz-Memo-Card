# 📚 Quiz Memo Card

Простое приложение-викторина с графическим интерфейсом, созданное на PyQt5.  
Проверяет знания, показывает правильные ответы и ведёт статистику.

## 🎯 Возможности

- ❓ Случайный выбор вопроса из списка
- 🔘 4 варианта ответа с перемешиванием
- ✅ Мгновенная проверка правильности
- 📊 Подсчёт количества правильных ответов и процента успеха
- 🔁 Автоматический переход к следующему вопросу


## 🛠️ Технологии

- Python 3.x
- PyQt5

## 🚀 Запуск

1. Установите PyQt5:
   pip install PyQt5

2. Запустите приложение:
   python sdfghj.py
   или
   готовый .exe
